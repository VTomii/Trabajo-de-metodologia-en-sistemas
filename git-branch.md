# Git branch
Concepto: El comando `git branch` tiene varias funcionalidades, ver el listado y la rama en la que estas ubicado, crear una nueva rama, eliminar alguna ya existente o renombrar una rama.

## Sintaxis: 
git branch (muestra con un `*` la rama donde estas ubicado)
git branch `nombre de la rama` (para crear una nueva rama)
git branch -d `nombre de la rama` (para eliminar una rama existente)
git branch -m `nombre de la rama` `nuevo nombre de la rama` (para renombrar una rama existente)
