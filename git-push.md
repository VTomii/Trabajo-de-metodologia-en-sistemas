# Git push
Concepto: El comando `git push` se utiliza para enviar al repositorio remoto los commits realizados en el repositorio local. De esta manera los cambios quedan disponibles para el resto de los integrantes del equipo. Es decir, una vez que a;adi los cambios con *git add* que vamos a subir al commit (*git commit*), enviamos al repositorio esos cambios para que luego se puedan sincronizar con el resto del contenido de este.

Sintaxis: 
git push
o
git push origin nombre-de-la-rama

en este trabajo por ejemplo, yo utilizo git push origin alumno-guerrieri para subir los cambios commiteados pero especificamente a mi rama
