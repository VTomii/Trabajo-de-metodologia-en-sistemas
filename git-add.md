# Git add
Concepto:
Utilizamos git add para añadir los cambios que vamos a subir con el commit. De alguna forma es dar a entender que las ultimas modificaciones que hicimos van a materializarse, porque los pasamos al "stage area" o zona de preparacion indicando que estan listos para el proximo commit.

##Sintaxis:
git add nombre-del-archivo
o 
git add .

## Ejemplo 
una vez que termine de modificar este archivo para hacer el commit primero debo hacer un git add . o git add git-add.md
