# Git checkout
Concepto: El comando `git checkout` se utiliza para cambiar a una rama existente, tambien agregando al final del comando `-b nombre-de-la-rama` se crea y se traslada a esa rama nueva en un solo paso.

## Sintaxis: 
git checkout `nombre-de-la-rama` (para moverse a una rama existente)
o
git checkout -b `nombre-nueva-rama` (para crear y moverse a una nueva rama)

## Ejemplo
En este caso, he utilizado `git checkout almuno-arrimada`, para hacer los commits en mi rama correspondiente.
