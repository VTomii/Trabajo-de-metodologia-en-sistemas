# Git log
El comando `git log` permite consultar el historial de commits realizados en un repositorio. Podemos conocer cambios fueron registrados, quien hizo esos cambios y en que momento se hicieron.
Eso permite que Git siga la evolucion del proyecto segun los cambios que se registran por lo integrantes del repositorio.
Segun la bibliografia, el log permite traer esas confirmaciones ya que el log recolecta la informacion importante como una biblioteca y te permite encontrar los libros mas importantes. *git log es esta indexación de tu biblioteca*

Sintaxis
git log

## Variantes
git log --oneline: muestra una version resumida del historial.
git log --stat: permite visualizar informacion sobre los archivos modificados en cada commit.
