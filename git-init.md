# Git init
Concepto:
El comando `git init` se usa para inicializar o activar un repositorio git dentro de una carpeta. Al ejecutarlo git crea una carpeta llamada ".git", donde almacena el historial del proyecto, las ramas y la configuracion necesaria para el control de versiones. 
Se utiliza al comenzar un proyecto nuevo que se quiere gestionar con Git.

Sintaxis:
git init

## Ejemplo
Al comenzar este trabajo practico, el primer paso fue inicializar el repositorio para que Git pudiera controlar los cambios del proyecto. Para ello, desde la terminal se ejecuto:
`git init`
De esta manera, la carpeta del proyecto paso a convertirse en un repositorio Git y quedo preparada para utilizar comandos como `git add`, `git commit` y `git status`.
