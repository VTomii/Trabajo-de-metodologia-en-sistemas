# git init
Concepto:
El comando `git init` se usa para inicializar o activar un repositorio git dentro de una carpeta. Al ejecutarlo git crea una carpeta llamada ".git", donde almacena el historial del proyecto, las ramas y la configuracion necesaria para el control de versiones. 
Se utiliza al comenzar un proyecto nuevo que se quiere gestionar con Git.

Sintaxis:
git init
