# Git pull

Concepto: El comando `git pull` permite traer al repositorio local los cambios realizados en el repositorio remoto. Esto permite que todos los integrantes tengan actualizado su "copia" del trabajo pudiendo observar la evolucion de este para poder continuar con la coherencia y para evitar trabajar en versiones desactualizadas.

## Sintaxis
git pull
o 
git pull origin alumno-guerrieri

ya que tambien se puede especificar el nombre de la rama de donde queremos traer los cambios
Por otro lado tenemos el git pull origin dev que permite obtener la version mas actualizada de las ramas que fueron compartidas a la dev
